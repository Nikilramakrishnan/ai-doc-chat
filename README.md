# AI Document Chat (Local, Free)

A web-based AI document chat application built with FastAPI and Ollama.

## Features
- Upload PDF or Excel files
- Ask questions in chat format
- Uses local LLM via Ollama
- No OpenAI / no paid APIs
- Fully offline

## Tech Stack
- FastAPI
- Ollama (local LLM)
- Pandas, PyPDF
- Python 3.9+

## How to Run

1. Start Ollama:
   ```bash
   ollama serve
